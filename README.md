# Settings

- モジュールのインストール

```bash
pip install -r requirements.txt
```

- APIキーの設定
  - config.cfg.sampleをコピーしconfig.cfgを作成
  - Qiitaアクセストークンを取得し、config.cfgの`BEARER_TOKEN`にセット

- アプリの起動

```bash
python main.py
```
